# Chatty-WS

A **real-time chat application** built with **Node.js**, **Express**, and **Socket.IO**, enabling instant messaging between multiple clients using **WebSockets**.

---

## **Demo / Screenshot**

![Chat Screenshot](talk.png)

> This screenshot shows the main chat interface of the app.

---

## **Features**

- Real-time messaging between multiple clients
- Instant message broadcast to all connected users
- Auto-scroll chat window for latest messages
- Send messages using Enter key or Send button
- Input validation and basic error handling

---

## **Installation**

1. Clone the repository:

```bash
git clone https://github.com/charuljain02/Socket-Talk.git
cd chatty-ws

2. Install dependencies:
npm install

3. Start the server
node index.js

4. Open your browser and go to:
http://localhost:9000

## Usage

Type your message in the input box and click Send or press Enter.
Messages are instantly broadcast to all connected clients.
The latest messages are always visible due to auto-scroll.
